- A.mp3
- B.mp3
- C.mp3
- D.mp3
- E.mp3
- F.mp3
- G.mp3
- H.mp3
- I.mp3

- M.mp3

- O.mp3
- P.mp3

- R.mp3
- S.mp3
- T.mp3
- U.mp3
- V.mp3
- W.mp3
- X.mp3
- Y.mp3
- Z.mp3

## Notes:
- Files should be in MP3 format
- File names are case-sensitive (use uppercase letters)
- If an audio file is missing, the app will fallback to text-to-speech
- Each audio file will be played repeatedly every 1 second until the spacebar is pressed
